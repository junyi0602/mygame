# Qt 正方形迷宫机器鼠项目报告

## 1. 项目概述
本项目是一款基于 Qt (C++) 框架开发的桌面端迷宫探索游戏。玩家将操控一只发条机械鼠，在未知的迷宫中进行探索，寻找通往终点的最短路径。游戏结合了路径记忆、多次尝试优化以及隐藏彩蛋机制，具有较高的趣味性和算法展示度。非常适合作为数据结构与算法（DFS/BFS）以及面向对象编程（Qt）的期末大作业或课程设计展示。

## 2. 核心玩法与规则
- **探索与记忆（战争迷雾）**：迷宫初始状态为全黑（未知），机械鼠移动时会自动感知周围的墙壁，并将其永久记录在全局地图中（已探索区域变为白色，墙壁变为亮橙色）。
- **两次机会机制**：
  - 玩家总共有两次机会。
  - 第一次机会主要用于探路，如果不小心撞墙则机会结束，回到起点。
  - 第二次机会可以使用第一次探索记录下的墙壁信息，规划最短路线冲刺。
- **星级评价系统**：游戏底层内置了 BFS（广度优先搜索）算法，会在生成迷宫时计算当前迷宫的绝对最短路径。玩家到达终点时的步数越接近理论最短步数，星级评价越高（最高为常规 3 星）。

## 3. 特色机制（隐藏彩蛋）
- **触发条件**：如果玩家在**第一次机会**中就以极高的效率到达终点并获得 **3星** 评价。
- **彩蛋效果**：在第二次机会开始时，迷宫的右上角会刷新出一个金黄色的“？”（神秘彩蛋格子）。
- **隐藏结局**：玩家在第二次机会中操控机械鼠到达该彩蛋位置，即可解锁隐藏的“五星级”最高评价与特殊的胜利提示。

## 4. 技术路线与实现细节
- **开发环境**：Qt 6 + C++17 + CMake
- **核心数据结构**：分离了 `MazeModel`（数据层）与 `GameController`（逻辑控制层），使得业务逻辑与视图解耦。
- **迷宫生成算法**：采用 **DFS（深度优先搜索）** 算法随机生成连通的迷宫，并随机打通部分墙壁以创造多条路径，避免只有单一死路。
- **最短路径算法**：采用 **BFS（广度优先搜索）** 算法计算从起点到终点的理论最少步数，作为星级评定的基准。
- **图形渲染机制**：通过重写 `paintEvent`，使用 `QPainter` 实现了迷宫格子、墙壁、发条机械鼠模型（`QPixmap` 平滑缩放）、终点红旗及彩蛋的定制化绘制。
- **UI 与交互**：采用 `QStackedWidget` 实现主菜单与游戏界面的无缝平滑切换；利用 Qt 的信号与槽（Signals and Slots）机制实现不同组件之间的通信。

## 5. 如何运行
1. 确保已安装 Qt Creator 和相应的编译套件（如 MinGW 64-bit 或 MSVC）。
2. 使用 Qt Creator 打开本项目的 `CMakeLists.txt`。
3. 点击“构建并运行”（或按 `Ctrl + R`）即可启动游戏界面。
