C++大作业实验报告
一.	作业题目
发条机械鼠迷宫
二.	开发软件
Qt Creator
三.	项目概述
本项目是一款基于 Qt 和 C++ 开发的 2D 桌面端迷宫探索游戏。玩家在游戏中扮演一只发条机械鼠，需要在一个 10x10 的随机迷宫中寻找通往蓝色红旗终点的最短路径。
在代码层面，本项目完整实现了以下几项核心游戏功能：
1.	每次开局利用算法随机生成独一无二、且保证有解的连通迷宫。
2.	迷宫初始为全黑状态，玩家必须操控机械鼠移动来“点亮”周围的地图，并永久记录下已经探测到的亮橙色墙壁。
3.	游戏限定玩家只有两次机会。第一次主要用于探路和背板；若撞墙则重置回起点（保留已探索的地图）。玩家需在第二次机会中利用记忆冲刺终点。
4.	游戏内置了寻路算法，会在后台自动计算当前迷宫的“绝对最短步数”。玩家通关所用的实际步数越接近这个最短步数，获得的星级越高。
5.	代码中埋设了特殊的触发逻辑，如果玩家在第一次机会中就完美达成 3 星，第二次机会的地图右上角将刷新一个神秘的“？”格子，触碰即可解锁隐藏的 5 星最高成就。
四.	主要流程
1.	搭建 Qt 工程骨架，实现 DFS 迷宫生成与基础网格数据结构的构建。
2.	加入探索机制、玩家键盘移动控制（WASD支持）以及基础的碰撞检测。
3.	实现 BFS 最短路径算法，接入两次机会机制与基于步数的等级评价系统。
4.	UI 界面美化，包括引入机械鼠图片素材、调整墙壁与背景颜色、加入开始界面与游戏规则弹窗。
5.	加入隐藏彩蛋机制。
五.	AI应用
在本项目的设计与开发过程中，我还使用了基于Gemini和ChatGPT大模型作为辅助工具。
AI的主要作用：辅助我完成对灵感构建实现思路；协助我快速搭建 Qt 工程的 CMake 骨架；在我描述了迷宫寻路的逻辑规则后，辅助我生成了部分 DFS 和 BFS 算法的基础代码片段；帮助我完成了关于界面设计，键盘捕捉等应用方面的代码；在处理由于键盘焦点冲突导致 WASD 按键失效等疑难 Bug 时，AI 提供了查错思路和解决方案；关于AI操作的一些代码，要求做了注释，以便于我理解和修改代码。
